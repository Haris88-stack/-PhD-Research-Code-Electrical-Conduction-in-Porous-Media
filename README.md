# -PhD-Research-Code-Mathematical-Modelling-of-Electrical-Conduction-in-Porous-Media
This repository contains code developed during my PhD research in Applied Mathematics, focused on the mathematical modelling and numerical simulation of electrical conduction in unconsolidated porous media.

## Overview

The project investigates how geometrical pore-scale models, together with volume averaging theory, can be used to predict the effective electrical conductivity of granular porous media. The work combines theoretical modelling with numerical simulations.

## Features

- Implementation of different pore configurations (e.g., arrays of types A1–A4, B1–B4, and C)
- Simulation of electrical conduction in porous structures
- Analysis of the impact of geometry and connectivity on conductivity
- Numerical simulations and validation studies

## Methods

The work is based on:
 - Mathematical modelling of transport phenomena in porous media
 - Numerical methods for solving partial differential equations
 - Finite volume methods implemented in MATLAB (coded from scratch, without using OpenFOAM or similar software)
